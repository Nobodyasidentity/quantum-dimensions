Quantum Dimensions

Quantum Dimensions is a Fabric mod by Nobodyasidentity that adds new dimensions, items, blocks and entities to Minecraft. Travel to the 27th Dimension or Hell, craft powerful tools, and survive new threats.

Minecraft 1.21.8 Fabric Fabric API required Java 21+ v0.0

Installation

Drop the .jar into your mods/ folder alongside Fabric Loader and Fabric API. Requires:

  • Minecraft 1.21.8
  • Fabric Loader ≥ 0.18.4
  • Fabric API
  • Java 21 or newer

Getting Started

The progression roughly goes:

  1. Gather 9 Blocks of Diamond and craft a Compact Block of Diamond.
  2. Craft a Quantum Machine from it — this is your dimensional teleporter.
  3. Place a Tesseract in the Quantum Machine's GUI to travel to the 27th Dimension.
  4. While in the 27th Dimension, craft a Rebreather to unlock the Subnautica advancement.
  5. Explore Hell — enter and exit to complete the To Hell and Back advancement.

An undead piglin hybrid that spawns in the Nether Wastes biome of Hell. Spawns in groups of 1–4 with a weight of 20. Holds a Golden Sword, is immune to fire, and has significant knockback resistance.

Unlike vanilla Zombie Piglins, this variant is always aggressive — it does not need to be provoked first.
